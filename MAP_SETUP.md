# 地图配置说明

## 问题
Google Maps 在中国无法正常访问，已替换为高德地图。

## 配置方式

### 方案一：使用高德地图 API（推荐）

1. **申请高德地图 API Key**
   - 访问：https://lbs.amap.com/
   - 注册/登录账号
   - 进入控制台 → 应用管理 → 创建新应用
   - 添加 Key，选择"Web端(JS API)"
   - 复制生成的 API Key

2. **配置环境变量**
   - 在项目根目录创建 `.env` 文件（如果不存在）
   - 添加以下内容：
     ```
     VITE_AMAP_KEY=你的高德地图API密钥
     ```

3. **重启开发服务器**
   ```bash
   npm run dev
   ```

### 方案二：使用备选方案（无需 API Key）

如果没有配置 API Key，页面会显示一个链接按钮，用户可以点击在新窗口打开高德地图查看位置。

## 注意事项

1. 高德地图 API 有免费额度，个人开发者通常足够使用
2. 如果 API Key 配置正确，地图会自动显示
3. 如果没有配置 API Key，会显示备选方案（链接按钮）
4. 地图支持缩放、拖拽等交互功能

## 测试

配置完成后，访问"联系我们"页面，应该能看到：
- 如果配置了 API Key：显示交互式地图，带有公司位置标记
- 如果未配置 API Key：显示地址和"点击查看地图位置"按钮
